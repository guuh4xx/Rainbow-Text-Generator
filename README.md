# texto-colorido

O `texto-colorido` é um formatador de texto, que pega um texto inserido e adiciona os famosos "color codes" do Minecraft, de uma forma que o texto fique rainbow.

**OBS:** O Minecraft Vanilla não suporta cores em placas e no chat por exemplo, então é necessário um plugin para que isso funcione.

![Resultado](https://i.imgur.com/LsA3ckH.png)

## Uso

Apenas baixe esse repositório e execute esse comando na raíz do projeto:

```bash
node .
```

Depois, coloque o texto que você quer que fique rainbow.

![Result](https://i.imgur.com/Rh4k4PI.png)
